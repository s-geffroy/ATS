Reference Python implementation

The ats.py module: bidirectional Gregorian ↔ ATS conversion, strict floor truncation, POSIX-aligned (leap seconds absorbed). Compatible with Python ≥ 3.9.

Download ats.py · View on GitHub

Quick usage

from datetime import datetime, timezone
from ats import gregorian_to_ats, ats_to_gregorian

now = datetime.now(timezone.utc)
ats = gregorian_to_ats(now)
print(ats.to_canonical())   # T+ Δ 20.7.8.0.61137
print(ats.to_short())       # Δ 20.7.8.0/61

back = ats_to_gregorian(ats.to_canonical())
print(back.isoformat())     # 2026-06-12T10:58:02+00:00 (±432 ms)
